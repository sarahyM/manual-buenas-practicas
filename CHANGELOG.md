# Historial de Cambios (Changelog)

Todos los cambios notables en este proyecto serán documentados en este archivo siguiendo las normativas de control de versiones.

---

## [2.0.0] - 2026-04-04

### Añadido
* Sección detallada sobre **Herramientas de Colaboración Digital** en el manual principal.
* Directrices de **Gestión de Tiempos** y productividad académica.
* Guía de **Estructura y Formato** para documentos universitarios.
* Estrategias de productividad integradas en el archivo README para acceso rápido.

### Cambiado
* Revisión integral del archivo README para mejorar la claridad, jerarquía y estructura técnica.
* Actualización del manual con mejores prácticas para proyectos grupales basadas en estándares de ingeniería.

### Corregido
* Ajuste de formato en los encabezados de `manual.md` para garantizar la consistencia visual en el renderizado de GitHub.
* Resolución de conflictos menores tras la fusión de ramas de colaboradores.

---

## [1.0.0] - 2026-04-03

### Añadido
* **Estructura base del repositorio**: Creación de carpetas `/documentos`, `/imagenes` y `/referencias` mediante archivos `.keep`.
* **Documentación inicial**: Creación del archivo `manual.md` y configuración del entorno de trabajo.
* **Investigación**: Incorporación de la sección de **Normas de Citación APA (7ma edición)**.
* **Repositorio**: Commit inicial y configuración de permisos para colaboración.
